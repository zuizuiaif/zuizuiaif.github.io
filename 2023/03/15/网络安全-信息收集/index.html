网络安全-信息收集

网络安全 2023-03-15

网络安全-信息收集

2023-03-15

网络安全

01真实IP获取

为什么要找到真实ip地址：找到真实的ip地址我们才可以访问到这个ip的C端和端口。通过ping可以获取到一个网站的ip地址。但是有的网站挂在了CDN，获取到的地址就不是真实地址，而是CDN地址。

那如何判断是否存在CDN呢：

1、使用不同主机ping域名判断是否存在CDN 站长工具 爱站网 国外ping探测

2、使用nslookup域名解析判断是否有CDN 用cmd命令行输入 nslookup baidu.com

那怎么绕过CDN获取到真实ip呢。

1、可以通过查询子域名的方式获取，因为 CDN非常的贵，很多网站只对主站做了CDN，子域名没有做CDN，而子域名和主站很大可能是在同一个服务器上。所以收集子域名ip就可以辅助获取判断主站的ip。

2、可以通过历史DNS解析记录查询到真实IP，通过查询DNS于IP绑定的历史记录就有可能发现之前的真实IP信息，DNSdb 微步在线

3、可以使用国外主机解析域名，部分国内CDN只做了国内的线路，但是国外线路没有做加速，这样就可以通过国外主机来探测到真实ip

4、网站漏洞，利用网站的漏洞和信息泄露的敏感文件信息，比如phpinfo文件网站源码文件，github文件等获取到真实ip

5、邮件信息，邮件中可能会记录邮件服务器地址，有些站点类似于RSS邮件订阅的功能，可以利用其发送邮件，通过查看源码的方式查看到服务器的真实ip

02旁站信息收集

旁站是与攻击目标在同一服务器上的不同网站，在攻击目标没有漏洞的情况下，可以通过查找旁站的漏洞攻击旁站，然后再通过提权拿到服务器的最高权限，拿到服务器的最高权限后攻击目标也就拿下了

获取旁站一般有这几个方式：

1、通过站长工具进行同IP的查询 https://stool.chinaz.com/same

2、通过bing搜索引擎查询 https://cn.bing.com/search?q=ip:x.x.x.x

3、通过网络空间探测引擎搜索 https://fofa.info/ 搜索 ip=”x.x.x.x”

03C段主机信息收集

C段查询主要是当前的IP攻击不下来的时候，可以攻击他的C段，比如IP是220.181.34.34是我们的目标，但是攻击不下来，可以去扫描220,181.34.0-255Z这个网段，看看能不能攻击下来

获取C段一般有这几种主要方式：

1、nmap扫描C段主机 nmap -sn x.x.x.0/24

2、利用google进行语法搜索 site:x.x.x.*

3、工具扫描 使用C段 工具扫描 x.x.x.0/24

4、使用网络安全探测引擎 ip=”x.x.x.0/24”

04子域名信息收集

子域名是父域名的下一级，像 “www.xxx.com ”这样的域名为企业的主站域名,企业对

于主站域名的应用的防护措施比较健全,不管是应用本身的漏洞发现、漏洞修复,还是安全

设备相关的防护都做得更加及时和到位,而企业可能有多个、几个甚至更多的子域名应用,

因为子域名数量多，企业子域名的防护可能没有主站的防护做的到位，攻击不到主站的时

候，尝试攻击子域名，进行子域名的信息收集。

1、枚举，枚举的方式就是进行暴力的破解，首先需要一个好的字典，这样会增加成功率。

2、使用google baidu bing 等搜索引擎搜索子域名。site:baidu.com

3、空间搜索引擎，使用fofa等工具搜索 domain=”baidu.com”

4、透明证书收集。通过crt网站查看 https://crt.sh/

5、聚合工具。聚合工具就是，结合和爆破，证书，搜索引擎，DNS解析记录等多个方式去探测子域名比

如：oneforall 使用 python oneforall.py –target baidu.com run

05端口信息收集

“端口”是英文port的意译，可以认为是设备与外界通讯交流的出口，不同的端口对应不同的服务，计算机的端口是0-65535个，当80 443 等端口攻击不下来的时候通过端口测试，扫描其他端口，从而渗透成功

1、使用nmap扫描端口 nmap -sV -p 1-65535 x.x.x.x

2、使用端口扫描的工具

这里提供四个软件

1、C段扫描器 webfinder

2、子域名挖掘机 Layer

3、子域名聚合收集工具 oneforall

4、端口信息收集工具 御剑
